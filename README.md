# 🌍 AirAware – Smart Air Quality Monitor

**Developed by RGAi Company** | **Created by Gauresh Khairnar**
*Environmental Studies & Sustainability Micro Project*

AirAware is a professional, responsive, and minimalist web platform that provides real-time Air Quality Index (AQI) data, promotes environmental awareness, and gamifies sustainable living.

## 🚀 Key Features
- **Live AQI Dashboard**: Real-time pollution data fetched via IQAir API.
- **Auto-Location**: Automatically detects your city to show local air quality.
- **Pollutant Breakdown**: Detailed levels for PM2.5, PM10, CO, and NO₂.
- **7-Day Forecast**: Weekly trends and hourly patterns for better planning.
- **Eco Actions Tracker**: Log green activities (cycling, planting trees) and earn "Green Points".
- **Dynamic UI**: Glassmorphism design, smooth scroll animations, and Dark/Light mode support.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
- **Animations**: CSS Keyframes + Intersection Observer API.
- **Charts**: Chart.js for interactive analytics.
- **API**: IQAir Visual REST API.
- **Icons**: Phosphor Icons.
